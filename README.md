# Portfolio

A premium personal portfolio built with React, Vite, and CSS Modules.

## Quick start

```bash
npm install
npm run dev
```

## Where to edit

| What | File |
|------|------|
| Name, role, tagline, nav | `src/content/site.js` |
| Hero copy and highlights | `src/content/site.js` |
| Colors, spacing, typography | `src/styles/tokens/` |
| Hero layout and styles | `src/sections/Hero/` |
| Shared buttons | `src/components/ui/Button/` |

## Sections

Each section lives in `src/sections/` with its own component and CSS Module.

Approved sections are locked — edit only when intentionally updating that section.
